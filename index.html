Scorpio4core

Runtime engine for Scorpio4

Download .zip Download .tar.gz View on GitHub

At it's heart, Scorpio4 is a semantic middleware platform, leveraging the expressive power of semantic web technologies (RDF, RDFS, OWL and SPARQL) RDF statement encode everything from data to behaviour, and express meaningful meta-data and relationships between any information, stored almost anywhere. The underlying RDF data model is expressive, self-describing, schema-free and web-scale - based on the technologies at the heart of the Semantic Web.
